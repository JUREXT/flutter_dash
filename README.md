# flutter_dash

A small library to draw dashed line in flutter :D

## Install

Just add `flutter_dash` as a dependency in your pubspec.yaml file.

## Usage

Here are some example:
```
Dash()
```
```
Dash(
  direction: Axis.horizontal,
  length: 132,
  dashLength: 12,
  dashColor: Colors.red)
  ```
